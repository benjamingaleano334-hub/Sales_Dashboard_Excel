# Proyecto: Dashboard de Ventas y Ganancias

Este proyecto presenta un **Dashboard interactivo en Excel** desarrollado a partir del archivo `sales_data.xlsx`.  
El objetivo es ofrecer una visión ejecutiva del rendimiento comercial mediante indicadores clave y visualizaciones dinámicas.

## 📊 Contenido
- `sales_data.xlsx`: Dashboard en Excel con KPIs y segmentadores.
- `analisis_sales.ipynb`: Notebook en Jupyter con explicación del dataset y conclusiones.
- `README.md`: Documentación del proyecto.

## 🚀 Características
- **KPIs principales**: Ingresos totales, Ganancias netas, Cantidad de órdenes y Producto más vendido.
- **Gráficos dinámicos**:
  - Ventas por categoría de producto.
  - Ventas por grupo etario.
  - Ingresos netos por país.
  - Evolución temporal de las ganancias.
- **Interactividad**: Segmentadores conectados a tablas dinámicas para filtrar por país, año y categoría.

## 🛠️ Tecnologías utilizadas
- **Excel**: Tablas dinámicas, segmentadores y visualizaciones.
- **Jupyter Notebook**: Documentación técnica y presentación del dataset

## 📈 Uso
1. Abrir `sales_data.xlsx` para explorar el Dashboard interactivo.
2. Revisar `analisis_sales.ipynb` para ver la explicación del dataset y las conclusiones.
3. Filtrar con los segmentadores en Excel para analizar distintas perspectivas.

## ✅ Conclusiones
El Dashboard permite explorar los datos de manera ejecutiva y clara, mientras que el Notebook aporta documentación técnica.  
En conjunto, forman un proyecto completo y profesional para portafolio.
